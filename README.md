# Spacex
&nbsp;
### Installation
   Ruby version 2.7.3
 
   Rails version 6.1.6
 
```sh
git clone https://github.com/jhjother/spacex.git
cd spacex/
bundle install
yarn install
```
```sh
rails s
```
>
>Access localhost from http://localhost:3000
>
